# Backend

Бэкенд-сервис для предварительной оценки ювелирных изделий. Обрабатывает запросы от фронтенда, интегрируется с AI-сервисом и создаёт сделки в Битрикс24.

## Технологии

- PHP 8.2
- Apache 2.4
- cURL

## Функциональность

- Приём и обработка запросов от фронтенда
- Интеграция с AI-сервисом (Python FastAPI)
- Создание контактов и сделок в Битрикс24
- Загрузка фотографий в сделку
- CORS поддержка
